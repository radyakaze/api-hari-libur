API Hari Libur

Karena rakyat suka tanggal merah

Cara mengakses API:

  • /api Untuk mendapatkan hari libur pada tahun ini.

  • /api?year=2023 Untuk mendapatkan hari libur pada pada tahun 2023.

  • /api?month=8 Untuk mendapatkan hari libur pada pada bulan ke-8 tahun ini.

  • /api?year=2023&month=8 Untuk mendapatkan hari libur pada pada bulan ke-8 tahun 2023.

Data API bersumber dari tanggalan.com.
